# Google AI Mode Chat → Markdown Export (Tampermonkey)

Export Google AI Mode chats to Markdown with one click.

## Features
- One-click export of chat sessions to a Markdown file
- Preserves code blocks
- Adds timestamps (when available)

## Installation

1. Install **Tampermonkey**:
   - Chrome/Edge: https://www.tampermonkey.net/
2. Create a new userscript in Tampermonkey.
3. Paste the script from `google_ai_export_markdown.user.js`.
4. Save the script.

## Usage
1. Open Google AI Mode (search results page).
2. Scroll to load all the chat content you want to export.
3. Click **⬇ Export AI Chat (MD)**.
4. The file `google_ai_chats.md` will download automatically.

## Notes
- Make sure you scroll to load all chat content before exporting.
- If the page layout changes, selectors may need updates.
